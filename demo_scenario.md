# BuyWise 시연 영상 시나리오 (3분 미만)

## 전체 구성: 2분 40초

---

## Scene 1: 오프닝 (10초)

**화면**: 앱 로그인 페이지
**나레이션/자막**:
> "매달 후회하는 구매, 얼마나 되시나요?"
> "BuyWise가 당신의 소비를 분석해드립니다."

---

## Scene 2: 로그인 (10초)

**화면**: Google 로그인 → 메인 화면 전환
**동작**:
1. "Google로 로그인" 버튼 클릭
2. 로그인 완료 → 메인 화면 진입

**자막**: "Google 계정으로 간편 로그인"

---

## Scene 3: 가계부 - 빠른 추가 (40초)

**화면**: 가계부 탭 (빠른 추가 폼)
**동작**:
1. 첫 번째 항목 입력 (충동 구매 예시)
   - 상품명: "무선이어폰"
   - 금액: 89,000
   - 카테고리: 전자제품
   - 고민기간: 0일 (충동 구매!)
   - 재구매의향: 아니오
   - 사용빈도: 2
2. "목록에 추가" 클릭
3. 두 번째 항목 빠르게 입력 (좋은 구매 예시)
   - 상품명: "운동화"
   - 금액: 120,000
   - 고민기간: 14일
   - 재구매의향: 예
   - 사용빈도: 5
4. "목록에 추가" 클릭
5. 대기 목록 2건 확인 → "모두 저장" 클릭

**자막**:
- "고민기간 0일 = 충동 구매 감지"
- "필요도가 자동으로 계산됩니다"
- "한 번에 여러 건을 저장할 수 있어요"

**포인트**: 필요도 자동 계산 박스가 변하는 걸 강조

---

## Scene 4: 누적 데이터 + 분석 실행 (15초)

**화면**: 내 구매 목록 (미리 샘플 데이터 20~30건 저장해두기)
**동작**:
1. 기간 필터 "전체" 확인
2. 저장된 구매 목록 스크롤 (간단히)
3. "누적 데이터 분석" 버튼 클릭

**자막**: "가계부 데이터가 쌓일수록 분석이 정확해져요"

---

## Scene 5: 분석 결과 - 차트 (30초)

**화면**: 분석 결과 (자동 스크롤)
**동작**: 위에서 아래로 천천히 스크롤하며 보여주기
1. **기본 통계**: 총 구매 건수, 총 지출, 평균 금액, 카테고리 수
2. **카테고리 차트**: 원형 차트 → 막대 그래프 전환
3. **필요도 vs 사용빈도 산점도**: 빨간 점(후회) vs 파란 점(만족) 대비

**자막**:
- "한눈에 보는 소비 패턴"
- "빨간 영역 = 후회 가능성 높은 구매"

---

## Scene 6: 후회 점수 (25초)

**화면**: 후회점수 분석 섹션
**동작**: 스크롤하며 보여주기
1. **후회점수 게이지**: 전체 점수 (예: 42/100)
2. **핵심 지표 4개**: 후회 비율, 후회 금액, 주요 원인, 만족 구매
3. **후회 TOP 10 테이블**: 빨간색 하이라이트된 항목들
4. **요인 분석 차트**: 7가지 요인별 막대 그래프

**자막**:
- "7가지 요소로 후회 점수를 계산합니다"
- "어떤 구매가 가장 후회되는지 한눈에!"

---

## Scene 7: AI 분석 (25초)

**화면**: AI 종합 분석 섹션
**동작**:
1. "AI 분석 시작" 버튼 클릭
2. 로딩 후 결과 표시
3. **심리 분석**: 소비 습관 요약, 후회 원인 TOP 3, 개선 방안
4. **스마트 인사이트**: 소비 패턴 분류, 추천 TOP 5, 절약 우선순위
5. **절약 시뮬레이터**: 슬라이더 조절 → 연간 절약액 변화

**자막**:
- "GPT-4o-mini가 당신의 소비 심리를 분석합니다"
- "맞춤형 절약 방안까지 제안!"

---

## Scene 8: 클로징 (5초)

**화면**: 앱 전체 화면 or 로고
**자막**:
> "BuyWise - 구매를 현명하게!"
> "buywiseapp.streamlit.app"

---

## 촬영 전 준비사항

### 1. 데이터 준비
- 미리 20~30건의 샘플 데이터를 가계부에 저장해두기
- 후회성 구매(고가+고민0일+사용빈도1)와 좋은 구매를 섞어서
- AI 분석도 한번 미리 실행해서 결과 캐싱해두면 로딩 시간 절약

### 2. 브라우저 설정
- 크롬 시크릿 모드 or 확장 프로그램 비활성화 (깔끔한 화면)
- 줌 레벨 100% or 110% (글씨가 잘 보이게)
- 북마크바 숨기기 (Ctrl+Shift+B)
- 탭 1개만 열어두기

### 3. 화면 해상도
- 1920x1080 (Full HD) 권장
- 브라우저 전체 화면 (F11)

## 촬영 도구 추천

### 무료 화면 녹화
| 도구 | 특징 |
|------|------|
| **OBS Studio** | 무료, 고품질, 자막 오버레이 가능 |
| **Windows 게임 바** | Win+G, 가장 간단, 바로 사용 가능 |
| **ShareX** | 무료, GIF/MP4, 영역 지정 녹화 |

### 편집 (자막 추가)
| 도구 | 특징 |
|------|------|
| **CapCut (PC)** | 무료, 자동 자막, 템플릿 풍부 |
| **DaVinci Resolve** | 무료, 프로급 편집 |
| **Clipchamp** | Windows 내장, 간단한 편집 |

### 가장 간단한 조합
**Windows 게임 바(녹화) + CapCut(편집/자막)** → 30분이면 완성

## 촬영 팁

1. **속도**: 클릭 사이 1-2초 여유 두기 (너무 빠르면 안 보임)
2. **마우스**: 클릭할 곳 위에서 잠깐 멈추기 (시청자가 따라갈 수 있게)
3. **실수 대비**: 3-4번 녹화해서 가장 좋은 것 사용
4. **AI 분석 로딩**: 미리 한번 실행 → 결과 있는 상태에서 녹화 (로딩 대기 시간 절약)
5. **나레이션**: 없어도 OK, 자막만으로 충분. 넣으면 더 좋음
