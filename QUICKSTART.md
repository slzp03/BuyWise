## 📤 3단계: CSV 업로드

### 방법 1: 샘플 데이터 사용
1. 왼쪽 사이드바에서 "📥 샘플 CSV 다운로드" 버튼 클릭
2. `sample_purchases.csv` 파일 다운로드
3. "CSV 파일을 선택하세요" 영역에 파일 업로드

### 방법 2: 직접 준비한 데이터 사용
프로젝트 폴더의 `sample_purchases.csv`를 참고하여 자신의 데이터 작성

**CSV 형식**:
```csv
날짜,카테고리,상품명,금액,필요도,사용빈도
2024-01-15,전자제품,무선이어폰,89000,3,5
2024-01-20,의류,겨울코트,150000,4,2
```

**필수 컬럼**:
- `날짜`: YYYY-MM-DD 형식
- `카테고리`: 텍스트
- `금액`: 숫자 (0 이상)
- `필요도`: 1-5 정수
- `사용빈도`: 1-5 정수

**선택 컬럼**:
- `상품명`: 텍스트 (없으면 카테고리명으로 자동 대체)

## 📊 4단계: 분석 결과 확인

업로드가 완료되면 자동으로 다음 분석이 표시됩니다:

### ✅ 표시되는 정보
1. **업로드된 데이터 테이블**
   - 전체 구매 내역 확인
   - 기본 통계 (총 건수, 총 금액, 평균 금액, 카테고리 수)

2. **카테고리별 분석**
   - 파이 차트 / 바 차트 (선택 가능)
   - 카테고리별 상세 통계 테이블

3. **추가 분석 차트**
   - 구매 금액 분포
   - 월별 지출 추이
   - 필요도 vs 사용빈도 산점도

4. **기본 인사이트**
   - 후회 구매 비율 및 금액
   - 합리적 구매 비율
   - 후회 가능성 높은 구매 TOP 5
   - 만족도 높은 구매 TOP 5

## 🛠️ 트러블슈팅

### 문제 1: 패키지 설치 오류
```bash
# pip 업그레이드
python -m pip install --upgrade pip

# 다시 설치
pip install -r requirements.txt
```

### 문제 2: CSV 업로드 오류
**에러**: "필수 컬럼이 누락되었습니다"
- 해결: CSV 파일에 필수 컬럼(날짜, 카테고리, 금액, 필요도, 사용빈도)이 모두 있는지 확인

**에러**: "날짜 형식이 올바르지 않습니다"
- 해결: 날짜를 YYYY-MM-DD 형식으로 변경 (예: 2024-01-15)

**에러**: "금액은 숫자여야 합니다"
- 해결: 금액 컬럼에 숫자만 입력 (쉼표, 원화 기호 제거)

**에러**: "필요도는 1-5 사이의 정수여야 합니다"
- 해결: 필요도와 사용빈도를 1, 2, 3, 4, 5 중 하나로 입력

### 문제 3: 한글 깨짐
- CSV 파일을 UTF-8 또는 CP949 인코딩으로 저장
- Excel에서 저장 시: "CSV UTF-8 (쉼표로 분리)" 선택

### 문제 4: 포트 충돌
```bash
# 다른 포트로 실행
streamlit run app.py --server.port 8502
```

## ❓ 자주 묻는 질문

**Q: 개인정보가 저장되나요?**
A: 아니요. 모든 데이터는 세션 기반으로 처리되며, 서버에 저장되지 않습니다.

**Q: 몇 건의 데이터가 필요한가요?**
A: 최소 3건 이상, 권장 10건 이상의 데이터가 있으면 더 정확한 분석이 가능합니다.

**Q: Excel 파일도 업로드 가능한가요?**
A: 현재는 CSV만 지원합니다. Excel에서 "CSV UTF-8"로 저장 후 업로드하세요.
